# 🏡 StayScape

### A Modern Airbnb Landing Page Clone

A responsive front-end clone of the Airbnb landing page designed to replicate the modern UI and user experience of Airbnb’s homepage.

Built to practice **modern UI layouts, responsive design, and front-end development.**

</div>

---

## 🚀 Project Overview

StayScape recreates the core visual structure of the Airbnb landing page including:

* 🔎 Hero search section
* 🏠 Property listing cards
* 🧭 Category browsing
* 📱 Responsive navigation bar
* 📲 Mobile-friendly design

This project focuses on building a **clean and responsive user interface** using modern front-end development techniques.

---

## ✨ Features

* Fully responsive Airbnb-style landing page
* Clean and modern UI design
* Property listing cards layout
* Navigation bar with search functionality UI
* Category browsing interface
* Mobile and tablet responsiveness

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/YOUR-USERNAME/stayscape.git
```

### 2️⃣ Navigate into the project

```bash
cd stayscape
```

### 3️⃣ Run the project

Simply open **index.html** in your browser.
